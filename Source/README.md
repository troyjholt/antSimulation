# antSimulation

To run:

Step 1 - Compile
Has to be compiled twice, like so.
------------------------
g++ -g -c main.cpp  
g++ main.o -o sfml-app -lsfml-graphics -lsfml-window -lsfml-system  

Step 2 - Run
------------------------
./sfml-app

A black screen should pop up. It is now waiting for you to start the simulation. 

Keys:
Num1 = Start
